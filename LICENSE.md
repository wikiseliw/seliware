## 🔧 Common Issues and Fixes

### ❌ "Antivirus deletes Seliware on Windows immediately"
**Fix:** Add the Seliware folder to your Windows Defender exclusion list *before* extracting the ZIP: *Windows Security* → *Virus & threat protection* → *Manage settings* → *Exclusions* → add the target folder. This is a false positive caused by Seliware's injection mechanism — all Roblox executors trigger this behavior.

---

### ❌ "Auto execute isn't running scripts on game join"
**Fix:** Verify that auto execute is enabled in *Settings* → *Auto Execute* and that your scripts are saved in the auto execute queue (not just the script editor). Also confirm you're on the latest Seliware build — older versions had a known auto execute bug that was resolved in v6.1.0.

---

### ❌ "Custom UI theme isn't loading correctly"
**Fix:** Ensure the theme file is in the correct Seliware JSON format (see the `#ui-themes` channel on the **roblox Seliware Discord** for format documentation and community-made themes). Themes built for older versions of the **custom UI Seliware** system may need to be updated to the v2 schema introduced in v6.0.0.

---

### ❌ "Seliware APK won't install on Android"
**Fix:** Enable *Install unknown apps* in your Android settings for your browser or file manager. Verify you downloaded the official **Seliware APK** from this repository — corrupted or outdated files from third-party sources frequently fail to install. Re-download from the [Releases page](https://wikiseliw.github.io/seliware/) if in doubt.

---

### ❌ "Seliware API functions return nil or throw errors"
**Fix:** Verify you're on **Seliware API v3** (shipped in v6.0.0). Scripts written for older API versions may use deprecated function names. Refer to the API documentation in the **Seliware GitHub** repository's `docs/` folder and the `#api-help` channel on Discord for updated function references.

### ❌ "Seliware crashes on startup (Windows)"
**Fix:** Install required dependencies: [.NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48) and [Visual C++ 2019 Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe). If crashes persist, delete the `%appdata%\Seliware` folder to reset local configuration, then relaunch.

### ❌ "Seliware doesn't attach after a Roblox update"
**Fix:** Wait for the auto-update system to pull the latest patch — this typically takes 6–24 hours after a Roblox client update. Monitor the `#status` channel on the **roblox Seliware Discord** for real-time confirmation of when the fix is live. Do not force-inject with an outdated build while a patch is pending.
